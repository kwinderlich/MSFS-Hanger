MSFS Hangar Release 155

Changes
- Made Explore Globe the primary map surface and removed the current/legacy map toggle from the UI.
- Moved MapLibre default navigation controls back into view and shifted custom globe controls away from them.
- Added top-center globe controls for Reset Globe, Center Current, 3D Terrain, 3D Buildings, Measure, and Range Rings.
- Moved the custom compass to the bottom-right and the ground-distance readout to the bottom-left.
- Marker hover now shows airport code and full airport name.
- Marker click now selects the airport and flies to a lighter first-stage zoom instead of over-zooming.
- Center actions now preserve the current zoom instead of zooming back out.
- Added dashed range rings for GA, turboprop, and jet ranges with a toggle.
- Reworked the measure tool so marker clicks stop selecting airports while measuring and the two-point line behavior works more like the MapLibre example.
- Added initial support for MapLibre vector-style base maps and 3D buildings at high zoom.
