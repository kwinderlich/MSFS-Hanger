MSFS Hangar Release 164

- Fixed Global Map airport/scenery resolution fallback so one failed resolver path no longer leaves the list and markers empty.
- Replaced brittle subtypeColor usage with safer marker-style color helpers.
- Added a visible custom suggestion dropdown for Range Aircraft in desktop mode.
- Default globe startup now opens over North America while the Region selector remains World.
- Active globe buttons now use dark blue backgrounds with yellow text/borders for better contrast over water and terrain.
- Preserves the last-used map style when opening Global Map instead of clearing the saved map state.
