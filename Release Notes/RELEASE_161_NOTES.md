MSFS Hangar Release 161

Explorer Globe fixes in this build:
- Preserve current map position/zoom when switching map style.
- Use globe camera state, not legacy Leaflet state, when saving/restoring map state.
- Fix marker query errors when airport hit layers are not yet present.
- Rebuild measure layers with valid MapLibre filters and persistent multi-point segments.
- Keep Reset Globe centered on the current map position while zooming out.
- Move altitude readout higher on the left side and compass higher on the right side.
- Remove the mapped-items summary line above the map.
- Improve Range Aircraft search with browser-native dropdown suggestions and clearer status text.
- Change "Other" airport subtype to a distinct orange color.
- Keep selected-airport Center in globe mode at the current zoom level.
