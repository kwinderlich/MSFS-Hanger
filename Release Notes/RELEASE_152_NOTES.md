MSFS Hangar Release 152

Focus: Explorer Globe

- Upgraded MapLibre frontend library to 5.23.0 for the globe path.
- Rebuilt Explore Globe around a stronger MapLibre globe-projection setup with atmosphere styling, larger ICAO labels, and always-visible marker dots.
- Marker clicks now select the airport as the current airport and fly toward a near-ground airport view.
- Added globe controls for Reset Globe, Center Current, Zoom In, Max Zoom, Zoom Out, 3D Terrain toggle, Measure toggle, compass, and distance-from-ground readout.
- Airport list Center / Zoom In / Max Zoom actions now use globe-specific zoom targets when the globe view is active.
- Increased map invalidation when switching back to the Current Global Map.
