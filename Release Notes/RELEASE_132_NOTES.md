MSFS Hangar Release 132

Highlights
- First-run bootstrap flow now asks for the initial base app folder before the backend starts.
- Active storage root is stored in a bootstrap config and used on future launches.
- Added library profiles with a default MSFS 2024 profile plus create/switch UI.
- Google search restored for browser-backed research and data-tab searches.
- Livery values parsed from aircraft.cfg now strip surrounding quotes.
- Packaging/output renamed for Release 132.

- Folder browsing now lets you create a new folder during selection and immediately use it for first-run setup or later app-folder changes.
