# MSFS Hangar Release 144

## Fixes
- Fixed Global Map crash by simplifying Leaflet initialization and tile-layer setup.
- Removed brittle map bounds options that could trigger immediate map failures.
- Improved built-in livery thumbnail detection inside the current aircraft package.
- Built-in livery texture folder resolution is now more tolerant of spacing/case differences.

- Explore Globe continues to use the stable world-map architecture with layers and airport markers.
