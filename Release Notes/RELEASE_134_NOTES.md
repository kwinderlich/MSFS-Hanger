MSFS Hangar Release 134

What changed:
- Added iniBuilds-specific livery scanning for separate livery packages and cabin packs.
- iniBuilds cabin packs now read presets/.../config/aircraft.cfg, use ui_variation for the displayed name, and prefer thumbnail/thumbnail_variation.png.
- iniBuilds liveries now read livery.cfg, pull the [General] name as the title, and prefer thumbnail/thumbnail.png.
- Livery cards now show the original livery package name.
- Added livery-grid filters for Active only and Favorites only.
- Added Activate Selected to the Liveries screen for bulk activation of selected livery packages.
- Continued stripping surrounding quotes from cfg-derived livery values.
- Created a dedicated Release Notes folder for per-build notes.
